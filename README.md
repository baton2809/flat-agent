# FlatAgent

AI-агент для помощи с вопросами недвижимости в России. Telegram-бот и REST API.

## Задача, для кого и какая боль

**Для кого:** физические лица, которые планируют покупку квартиры - первичка или вторичка, с ипотекой или без.

**Боль сейчас:**
- Расчёт ипотеки размазан по разным банковским калькуляторам, каждый из которых выдаёт разные цифры
- Актуальную ключевую ставку ЦБ нужно искать отдельно
- Сравнение вариантов (новостройка vs вторичка) требует ручной агрегации данных из Циан/Авито
- Нет единого места, где можно задать вопрос о рынке, юридических нюансах и тут же посчитать ипотеку

**Что делают сейчас:** открывают 4-5 вкладок браузера, копируют данные между ними вручную, теряют контекст между сессиями.

## Что именно сделает PoC на демо

1. Пользователь пишет в Telegram: "посчитай ипотеку 8 млн на 20 лет с первоначальным взносом 20%"
2. Агент тянет актуальную ключевую ставку из ЦБ РФ, считает аннуитет, выдаёт ежемесячный платёж и итоговую переплату
3. Пользователь загружает прайс-лист застройщика (спарсенный с сайта) — агент автоматически распознаёт структуру, строит OLS-регрессию цена/площадь, выявляет выбросы и выдаёт визуализацию с аналитическим выводом
4. Агент помнит бюджет и предпочтения пользователя между сессиями (долгосрочная память в SQLite)
5. Поиск актуальных объявлений и новостей рынка через DuckDuckGo с суммаризацией GigaChat

## Что НЕ делает PoC (out-of-scope)

- Не подключается напрямую к базам ЦИАН/Авито через официальные API (нет доступа)
- Не оформляет заявки на ипотеку и не взаимодействует с банками
- Не верифицирует юридическую чистоту объекта (выписки ЕГРН, обременения)
- Не поддерживает голосовые сообщения
- Не работает с коммерческой недвижимостью
- Не поддерживает мультипользовательские сценарии с общим состоянием
